# MERN Integration Project - Entri Elevate CRM

**Educational and Practice Purpose Only**
**Developer:** Ashfaaq Feroz Muhammad - Entri Elevate MERN

## Overview
This is a Customer Relationship Management (CRM) application built using the MERN stack (MongoDB, Express.js, React.js, and Node.js). It includes user authentication, full CRUD operations for managing customer data, database connectivity logic, separate modular routing/controllers, responsive React UI using **Tailwind CSS**, and proper error handling.

## Features
- **Frontend**: Built with React and Vite. Features a visually stunning custom Blue-to-White gradient theme and fully functional Dark/Light mode support.
- **Backend**: Built with Node.js and Express. Modular folder architecture utilizing Controllers, Routes, and Middleware.
- **Authentication**: Secure JWT-based registration and login system.
- **Database**: MongoDB integration using Mongoose models (Users and Customers) with validation.
- **CRUD Operations**: Secure endpoints for managing customer data properly authorized via JWT tokens.
